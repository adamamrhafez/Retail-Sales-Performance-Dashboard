# Retail Sales Performance Dashboard

A comprehensive Power BI dashboard designed to analyze global retail sales performance, providing deep insights into profitability, channel efficiency, and regional trends.

## 📊 Dashboard Overview
This project visualizes key retail metrics to support data-driven decision-making. It covers performance across multiple countries, product categories, and sales channels.

## 🚀 Key Metrics & Insights
* **Total Sales:** 2.05M
* **Sum of Profit:** 443.72K
* **Order Volume:** 8.762K
* **Category Performance:** Electronics leads as the top-performing category in terms of profit.
* **Sales by Channel:** Channel 3 contributes the highest share of total sales (57%).

## 🛠️ Tools & Technologies
* **Power BI:** Data modeling, DAX, and visualization design.
* **Data Analysis:** Tracking shipping efficiency vs. sales trends and analyzing the impact of discounts on profitability.

## 🌍 Global Coverage
The dashboard allows for interactive filtering by year (2022-2025) and various countries including Egypt, Saudi Arabia, UAE, Kuwait, Jordan, Qatar, Tunisia, and Morocco.
